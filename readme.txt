簡單設計一個class 裡面有push和pop
可以利用這個class來記錄路徑
任務一 就是上下左右走訪，走到死路往回走，直到找到終點
任務二 就是上下左右走訪，並隨時記錄找到寶藏的數量，直到找到所有寶藏
       如果沒找到足夠數量，會走遍所有路徑，然後回到原點
任務三 就是上下左右走訪，並在走過的路徑上標記權重，邊走遍所有路徑邊更新最短路徑。
